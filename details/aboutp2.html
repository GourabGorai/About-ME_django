Flask Cryptocurrency Price Prediction Application

This Flask application allows users to fetch historical cryptocurrency data, train a prediction model, and predict future prices.

Features

  • Fetch Data: Retrieve historical data for a specific cryptocurrency symbol over a specified date range.
  • Train Model: Train a linear regression model using historical data to predict future prices.
  • Predict Price: Predict the future price of a cryptocurrency based on the trained model and a specified future date.

Endpoints

Index

GET / - Render the main page.

Fetch Data

POST /fetch_data - Fetch historical data for a given cryptocurrency symbol, start date, and end date.

{
    "symbol": "BTC",
    "start_date": "2023-01-01",
    "end_date": "2023-12-31"
}

Train Model

POST /train_model - Train a linear regression model using historical data from 2023 and 2024.

{
    "symbol": "BTC"
}

Predict Price

POST /predict_price - Predict the future price of a cryptocurrency for a specified future date.

{
    "symbol": "BTC",
    "user_date": "2024-12-31"
}

Technologies Used

  • Flask: Web framework for building the application.
  • Requests: Library for making HTTP requests to the CryptoCompare API.
  • Pandas: Library for data manipulation and analysis.
  • Scikit-learn: Library for machine learning, used for training the linear regression model.
  • Matplotlib: Library for plotting and visualizing data.
  • Gunicorn: WSGI HTTP server for running the Flask application in production.
